## ‚ú® Features

-   **Hybrid Pro Face**: A stunning combination of 3D analog mechanics and digital precision.
    -   *Skeletal Mode*: Peek inside the movement at the gears and quartz oscillator.
    -   *Atomic Sync*: Simulates NTP synchronization for ultimate accuracy.
    -   *Smart Sensors*: Displays environmental data (Temp/Humidity).

    -   **Alarms**: Reliable, persistent alarms with custom digital soundscapes.

    -   **Day Mode**: Clean #212329 Gray aesthetic (OLED friendly).
    -   **Night Mode**: Deep Red preservation mode to protect your night vision and prevent burn-in.

    -   Resizable clock face.
    -   Brightness controls.
